Common HTML Tags

These are the most commonly used HTML tags and a short description of each

Block Elements

Block elements take up their own line and take up the full width of the page

body

This tag defines the document's body

head

This tag defines information about the document

html

This tag defines the root of an HTML document

div

This tag defines a section in a document

section

This tag defines a section in a document

header

This tag defines a header for a document or section

main

This tag specifies the main content of a document

article

This tag defines an article

nav

This tag defines navigation links

footer

This tag defines a footer for a document or section

p

This tag defines a paragraph

aside

This tag defines content aside from the page content

li

This tag defines a list term



Inline Elements

Inline elements only take up as much space on a line as they need to. They do not start new lines

a

This tag defines a hyperlink

strong

This tag defines important text

em

This tag defines emphasized text

img

This tag defines an image

span

This tag defines a section in a document
